# Dropkick
Introducing a highly opinionated Content Management System,
(EXPERIMENTAL) - Drop Kick GO!

## Goals
- Separation of Concerns between Business Logic and Templating
- Dynamic content creation
- Ease of USE

## Requirements
- Go
- MongoDB (Optional)

## Command Line Installation
- `go run app.go`
- open localhost:8080

## Install with Go
- `go install`
- `dropkick`

## Command Line Help
- `dropkick -h`
